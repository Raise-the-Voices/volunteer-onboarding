# Set up a meeting (a real one)

Sharing a calendar event in Slack is a skill you'll use constantly here — for
team calls, interviews, and outreach. So let's learn it by doing a **real** one,
not a pretend one.

You'll also practise a move we use all the time:
**find one person who'll do it with you, then open it to everyone.**

**Goal:** create a real event on the RTV calendar, get its shareable link, and
open it to anyone interested.

- [ ] Pick a **real topic** you'd genuinely like to talk about — an issue, a country, or a skill to learn together
- [ ] Find **one partner**: message someone in **#general** or a country/area channel and agree on a time
- [ ] Create the event in Google Calendar, then **copy it onto the RTV calendar** (open the event → three-dot menu → *Copy to…* → the RTV calendar)
- [ ] Give it a clear name — someone seeing only the title should know what it is
- [ ] Open the event, choose **Publish event**, and **copy the link**
- [ ] Post that link in **#general** (or your country/area channel): say what it's about and that **anyone interested is welcome to join**

**Done when:** your event is on the RTV calendar and the link is shared for anyone to join.

> This is the **"one, then all"** move: you never have to organise alone. Get
> one *yes*, then open the door to everyone. It works for meetings, actions, and
> whole campaigns.
>
> Clues: look at existing events on the RTV calendar for the naming pattern.
