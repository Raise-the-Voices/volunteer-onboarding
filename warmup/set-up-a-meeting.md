# Set up a meeting

Setting a meeting, inviting people, and sharing the link is a skill you'll
use constantly here — for team calls, and later for outreach.

**Goal:** create a calendar event, invite someone, and share the link in the chat.

- [ ] Create an event in Google Calendar — a 15-minute intro call with another new volunteer is ideal
- [ ] Give it a clear name — would someone seeing only the invite know what the meeting is?
- [ ] Invite at least one other person
- [ ] Share the event link in the volunteer chat

**Done when:** someone else opens your link and sees the event.
