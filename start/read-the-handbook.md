# Read the handbook

**Goal:** understand how Raise the Voices works before you dive in — it makes
every task after this make sense.

- [ ] Read the [Volunteer Handbook](https://docs.google.com/document/d/14Kf-bJidXohb2FO0RVgDx6ZGyI08sgEHw6vAUzfYif8/edit) — especially **How We Work**
- [ ] Learn **the Learning Cycle** — it's how we work every case:
  1. **Understand** — read up on the situation, and write a short *backgrounder* so others get the big picture
  2. **Reach out** — talk to the people affected, directly
  3. **Take action** — publish the interview, write, lobby, advocate
- [ ] Notice the four things we make: **backgrounders, interviews/articles, newsletters, and private outreach**
- [ ] Pick the project or region that pulls at you most — you'll use it in the next tasks

**Done when:** you could explain the Learning Cycle to another volunteer in one breath.

> Everything you do here is real. Even while you're still learning, you're
> already helping — so pick something you genuinely care about. Your time is
> valuable, and we don't waste it on pretend exercises.
