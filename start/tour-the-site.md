# Tour the site

**Goal:** know what Raise the Voices shows the world.

- [ ] Open [raisethevoices.org](https://raisethevoices.org) and read the front page
- [ ] Read one story all the way through
- [ ] Write down one thing that surprised you, or that you want to know more about — you'll use it in [Pick your issue](/warmup/pick-your-issue.md)

**Done when:** you can tell another volunteer, in a sentence or two, what Raise the Voices does.
