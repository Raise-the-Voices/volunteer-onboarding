# Source docs

Dump docs here (Google Docs exports — .md, .txt, .docx, whatever), then ask
Claude to run `/update-tasks`. It turns them into task pages, or uses them to
fill existing `[FILL IN]` gaps, keeping your wording.

Everything in this directory except this README is **gitignored** — source
docs may contain private contacts and internal process, so they stay on this
machine and are never published or committed.
